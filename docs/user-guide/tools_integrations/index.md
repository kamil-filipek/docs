# Integrations and Tools

Tools and integrations extend the functionality of AI/Run CodeMie. Tools are features that enable your Assistant to perform different tasks. While some tools operate independently within the platform, most tools require integration with external systems.

Integration is a secure way to store credentials and necessary information for further connection to external services or systems. They allow your Assistant to interact with third-party platforms such as code repositories, Jira, email, and more. By storing access credentials (e.g., URLs, tokens) or any other settings encrypted securely, integrations ensure that your Assistant can perform tasks requiring authentication without exposing critical information.

## Overview

This section covers:

### [Integrations](./integrations/index.md)

Learn how to set up and manage integrations with external systems to enable tool functionality.

### [Tools](./tools/overview.md)

Configure and use various tools to extend assistant capabilities:

- **Agent Collaboration**: A2A Protocol for connecting external AI agents and cross-framework interoperability
- **Authentication & Security**: Keycloak integration for user authentication
- **Code Quality**: SonarQube for code analysis and quality metrics
- **Database**: SQL tools for database operations
- **Communication**: Telegram and Email Sender for notifications and messaging
- **Cloud Platforms**: Azure, GCP, and AWS integrations for cloud operations
- **DevOps**: Kubernetes for container orchestration, Webhook for event triggers
- **Monitoring**: Elastic for logging and monitoring
- **Development**: Codebase Tools for code analysis, OpenAPI Tool for API documentation
- **Project Management**: Jira, ServiceNow, Xray for task and test management
- **Quality Assurance**: Jira Zephyr Scale integration for test management
- **Automation**: Scheduler for automated tasks, Plugin framework for custom tools
- **Version Control**: Git integrations (GitHub, GitLab, Bitbucket, Azure DevOps)
- **MCP**: Model Context Protocol server integration

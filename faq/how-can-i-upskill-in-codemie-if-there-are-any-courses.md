# How can I upskill in CodeMie if there are any courses? Where should I start learning CodeMie? Are there practical exercises or tasks in CodeMie? What learning materials are available for CodeMie? Are there structured courses for CodeMie? What can be offered to a client along with the CodeMie platform? Do I need to contact the CodeMie team for training?

There are several ways to learn and develop skills on the **CodeMie** platform.

It is recommended to start with the **Onboarding course**, which helps you quickly get familiar with the platform, its features, and common usage scenarios:  
[https://www.youtube.com/playlist?list=PLJrHeCAQNQd6EXbhDoRQW3GKnUpWe7day](https://www.youtube.com/playlist?list=PLJrHeCAQNQd6EXbhDoRQW3GKnUpWe7day)

To practice and reinforce your skills, **CodeMie Katas** are available — practical exercises designed to help you work hands-on with the platform:  
[https://codemie.lab.epam.com/#/katas](https://codemie.lab.epam.com/#/katas)

Additionally, **L\&D training courses** are available, offered **as an optional addition in coordination with the CodeMie team**.  
To gain access, please contact the CodeMie team:

- AI/Run CodeMie Basic Course  
  [https://epa.ms/codemie-course-basic](https://epa.ms/codemie-course-basic)
- AI/Run CodeMie Advanced Course  
  [https://epa.ms/codemie-course-advanced](https://epa.ms/codemie-course-advanced)

When providing the platform to a client, it is recommended to include the **Onboarding course** and **Katas** as the basic set.

## Manage Katas

Create, edit, publish, clone, and archive AI Katas.

This functionality is only available to users with the isAdmin role. Platform Administrators have full access to create, edit, publish, and manage all Kata content.

# How can I process images embedded in PDF documents? Does CodeMie extract and analyze images from PDF files?

## PDF Processing Capabilities in CodeMie

CodeMie provides comprehensive processing of PDF documents, handling both text and images embedded within these files. This functionality is available through both the /model endpoint API and as chat attachments.

## Features

- **Text Processing**: Extracts and analyzes all text content from PDF documents
- **Image Processing**: Identifies, extracts, and processes images embedded within PDF documents
- **Combined Analysis**: Provides context-aware analysis that understands the relationship between text and images in the document

## How to Use

Via Chat Attachment

1. Click the paperclip button in the chat interface
2. Upload your PDF document (up to 50MB)
3. Ask questions about both the textual and visual content of the document

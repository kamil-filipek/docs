# What file types can I specify for adding/indexing repositories?

You can specify file extensions (e.g., ".java", ".yaml", ".js") or exact file names (e.g., "README.md, build.gradle, .groovy"). If you leave this empty, all files are indexed, which might include non-relevant files.

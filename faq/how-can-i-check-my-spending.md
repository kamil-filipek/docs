# How can I check my spending? How can I track token usage? Where can I view project analytics? How can I see how many tokens or how much money my team has spent?

The Analytics Dashboard provides comprehensive monitoring of LLM usage, costs, and activity metrics across the CodeMie platform. Access to data varies based on your role and permissions.

### Accessing the Analytics Dashboard

1. Click on your **Profile icon** in the bottom left corner
2. Select **Settings** from the menu
3. Click **Analytics** in the Settings navigation menu

The dashboard displays real-time insights into token consumption, spending patterns, and system utilization.

### Data Visibility by Role

**Regular Users**

- View personal statistics only (your own usage and costs)
- Monitor your individual assistants, workflows, and tool usage

**Admin Role (isAdmin)**

- Access complete platform-wide analytics
- View statistics for all projects and users in the organization
- Monitor all assistants, workflows, tools, and integrations across the platform

**Project Admin Role (applications_admin)**

- View statistics only for projects where you are a Project Admin
- Monitor usage and costs for all users within your managed projects
- Access project-level metrics for assistants and workflows

> **Note**: To view analytics for other projects or users, you need either the isAdmin role or applications_admin permissions for specific projects. Contact your administrator if you require broader access.

### Using the Filtering System

The Analytics Dashboard includes a global filtering system that refines data across all sections:

**Time Period Filter**

- Quick presets: Last Hour, Last N Hours, Last N Days, Last Month
- Custom Range: Select specific start and end dates with time

**User Filter**

- Multi-select dropdown with autocomplete
- Filter by specific users or select "Me" for your activity
- Combine multiple users for comparison

**Project Filter**

- Multi-select dropdown with autocomplete
- View project-specific usage and costs

**Filter Behavior**

- All active filters work together to refine data
- Filter selections persist as you navigate between dashboard sections
- Use "Clear all" button to reset all filters at once

Access Permissions and Data Visibility  
The Analytics Dashboard data visibility depends on your user role and permissions defined in JWT attributes:

Regular Users  
Regular users without admin privileges:

View only personal statistics (your own usage and costs)  
Access data related to your individual activity  
Monitor your own assistants, workflows, and tool usage  
Admin Role (isAdmin)  
Users with the isAdmin role have full access to analytics data across the entire CodeMie platform:

View statistics for all projects in the organization  
View statistics for all users using CodeMie  
Access complete platform-wide usage and cost metrics  
Monitor all assistants, workflows, tools, and integrations  
Project Admin Role (applications_admin)  
Users with applications_admin permissions have limited access to analytics data:

View statistics only for projects where you are a Project Admin  
View data for all users within your managed projects  
Monitor usage and costs specific to your projects  
Access project-level metrics for assistants and workflows

# How do I create an A2A Remote Assistant?

To create an A2A Remote Assistant in CodeMie:

1. Navigate to Assistants and click the Create Remote Assistant button in the top-right corner
2. On the Create Remote Assistant page, configure the basic settings:
   - Project: Select the project where the assistant will be created
   - Share with project: Toggle on to make the assistant available to project team members
   - Integration (Optional): Leave empty for public agents
3. Enter the Assistant URL (e.g., https://example.com/agent or https://example.ngrok-free.dev) and click Fetch
4. After clicking Fetch, CodeMie will send a request to retrieve the agent card
5. Review the fetched agent card details containing assistant name, description, skills, and other metadata
6. Click Save to create the remote assistant

The assistant card provides transparency about what the remote agent can do. Review it carefully to understand the agent's capabilities before saving.

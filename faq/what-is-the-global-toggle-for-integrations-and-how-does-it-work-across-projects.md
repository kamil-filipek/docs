# What is the Global toggle for integrations and how does it work across projects?

## Global integrations Toggle Feature

## What is the Global Toggle?

The Global toggle is a new feature that allows you to make specific user integrations accessible across every project you manage, eliminating the need for additional setup in each project.

## How to Use Global Integrations

1. Navigate to the \***\*User Integrations\*\*** menu
2. Find the integrations you want to make global
3. Enable the \***\*"Global"\*\*** toggle switch
4. The integrations will now be available across all your projects

## Benefits

- \***\*Cross-project availability\*\***: Use the same integrations settings across multiple projects
- \***\*Simplified management\*\***: No need to recreate integrations for each project
- \***\*Consistent configuration\*\***: Maintain uniform integrations settings organization-wide

integrations Selection in Marketplace

- You can \***\*select your own integrations\*\*** from your personalized list

- Global integrations appear in all relevant integrations selectors

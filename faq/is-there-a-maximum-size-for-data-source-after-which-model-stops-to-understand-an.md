# Is there a maximum size for Data Source after which model stops to “understand” and loses context of it? The model has a context window, but it is irrelevant to Data Source size. The model answer will depend on query quality. There is a use case when the model has thousands of Data Sources, however with queries that are specific and on point there are no problems. If a poor query provided e.g., “tell me about something” answer would be vague.

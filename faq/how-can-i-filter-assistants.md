# How can I filter assistants? Can I filter assistants by project? How can I filter assistants at the All Assistant page?

You can filter assistants in several ways:

1. **By name or project**: Go to the "Assistants" tab and enter either the assistant's name or the project name in the search bar and press the search button.

2. **By sharing status**: On the 'All Assistant page', you can use the status filter dropdown to filter assistants based on their sharing status:
   - **Shared with project**: Shows assistants that are shared with your project team

   - **Not shared with project**: Shows assistants that are personal and not shared with the project

   - **Global Assistant**: Shows assistants that are available globally across the platform

The selected filter status will be saved on the UI and persist even after page reloads or navigation.

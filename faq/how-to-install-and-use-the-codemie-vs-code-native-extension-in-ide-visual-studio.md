# How to Install and Use the CodeMie VS Code Native Extension in IDE Visual Studio Code?

## Overview

The CodeMie VS Code extension brings native CodeMie integrations directly into Visual Studio Code. This extension provides a dedicated sidebar interface with full-featured access to CodeMie assistants, conversations, and tools without leaving your editor.

## Prerequisites

- **Visual Studio Code** (latest version recommended)
- **CodeMie Account** for authentication

## Installation

1. Open your Visual Studio Code IDE
2. Click on the **Extensions** button or press **Ctrl+Shift+X** (Windows/Linux) or **Cmd+Shift+X** (macOS)
3. Search for **"AI/Run CodeMie"** in the extensions marketplace
4. Click **Install** and follow the setup instructions

## Key Features

### Native IDE Integration

- **Sidebar Interface**: Access CodeMie directly from the VS Code sidebar
- **Project Context**: Automatic integrations with your current development context
- **Assistant Tools**: All development tools are automatically integrated when selecting assistants

### Conversation Management

- **Chat History**: Navigate through previous conversations with assistants
- **New Conversations**: Create new assistant interactions as needed
- **Web Sync**: All IDE conversations are synchronized with the CodeMie web platform

## Troubleshooting

### Common Issues and Solutions

#### Extension Not Responding

1. Click the **Refresh** button in the extension interface
2. Restart your Visual Studio Code IDE
3. Check connection status

#### Authentication Problems

1. Verify connection
2. Clear extension cache in settings
3. Reset login data through extension settings

### Extension Settings

Access extension settings to:

- **Toggle Toolkits**: Enable/disable specific development toolkits
- **Add MCP Plugins**: Configure Model Context Protocol plugins
- **Reset Cache**: Clear cached data and login information
- **Manage Connections**: Update authentication and connection settings

You can also use **codemie-cli** for development.

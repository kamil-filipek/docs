# What is an A2A Agent Card and what information does it contain?

An A2A Agent Card is a metadata description of an agent's capabilities that CodeMie fetches when you create a remote assistant. The agent card contains:

- Assistant name: Pre-populated from the agent (can be customized)
- Description: Agent's purpose and capabilities
- Skills: Available tools and functions
- Other metadata: Version, supported modes, etc.

The assistant card provides transparency about what the remote agent can do. Review it carefully to understand the agent's capabilities before saving.

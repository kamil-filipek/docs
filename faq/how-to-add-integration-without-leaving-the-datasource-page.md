# How to add integration without leaving the datasource page? Can I create integrations directly from a datasource view? Where is the Add Integration button on the datasource page?

1. Navigate to your datasource page where you have existing integrations listed
2. Locate the "Add Integration" button - This button appears on the page and follows the same UI pattern as the tools section in the assistant creation form
3. Click the "Add Integration" button to launch the integration creation workflow
4. Configure your new integration using either:
5. Modal dialog that opens on the same page
6. Inline creation form embedded in the page
7. Complete the integration setup by providing:
8. Integration alias
9. Required credentials (URL and token)
10. Any additional configuration parameters
11. Save the integration - The new integration will appear immediately in your existing integrations list

# How can I create an assistant from a template in CodeMie?

To create an assistant from a template in CodeMie, follow these steps:

1. On the main page, click the "Explore Assistants" button.
2. In the "Explore Assistants" page, select the "Templates" tab.
3. Choose the desired assistant and click it.
4. In the "Create Assistant" menu, scroll down and click "Create".
5. The assistant will then appear in the Assistants -> Project Assistants menu.
6. Click the assistant’s name to start a conversation.

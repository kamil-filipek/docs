# System instructions? Instructions history? System instructions version rollback? System instructions version management?

When working with assistants, you might experience different outcomes depending on the system instructions you use. The latest instructions don’t always produce the best results, so you may want to revert to earlier versions. The challenge lies in remembering which instructions were more effective. This is where the "system instructions history" comes in handy. It allows you to save all the instructions you've set for the assistant, making it easy to select and reuse them later if needed. To revert to a previous version of system instructions, follow these steps:

1. Go to the assistant you want to revert and click "Edit."
2. In the assistant’s edit menu, select the "History" tab.
3. Click the version title to choose the version you need.
4. Select one of the previous versions and click the "Update" button.

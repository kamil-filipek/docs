# How do I set up Azure DevOps integration in CodeMie?

To connect CodeMie to Azure DevOps, go to **Integrations**, click **+ Create**, and select
**AzureDevOps** as the Credential Type. Fill in your organization URL
(e.g., `https://dev.azure.com`), Organization Name, Project Name, and a Personal Access Token
(PAT) with the required scopes (Work Items, Wiki, and/or Test Management read & write).

Once saved, the integration becomes available for selection when configuring Azure DevOps tools
in any assistant.

# How does A2A relate to Sub-Assistants and Multi-Assistant Orchestration in CodeMie?

A2A enables agent-to-agent communication both with external agents and within CodeMie. For more information about agent-to-agent communication within the CodeMie platform itself, see the Sub-Assistants and Orchestration documentation.

The A2A protocol provides the foundation for agents to communicate and collaborate, whether they're external agents integrated via A2A Remote Assistants or internal CodeMie assistants working together.

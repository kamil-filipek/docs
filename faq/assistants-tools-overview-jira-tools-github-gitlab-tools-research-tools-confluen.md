# Assistants Tools Overview. Jira tools. GitHub, GitLab tools. Research tools. Confluence tool.

**Q: What tools are provided in CodeMie for assistant enhancements?**  
A: CodeMie provides four major tools for assistant enhancements: Jira, Git, and Research.

**Q: What capabilities does the Jira toolkit offer?**  
A: The Jira toolkit offers several capabilities including creating Jira issues, searching for Jira issues, getting Jira projects, updating Jira issues, and adding comments to a Jira issue.

**Q: How can the Git toolkit assist me?**  
A: The Git toolkit assists by setting active branches, creating files, updating files, creating pull requests, creating branches, linking branches in a repository, getting changes of a pull request, and creating pull request change comments.

**Q: What is the purpose of the Confluence toolkit?**  
A: Using the Confluence toolkit you can search relevant information and pages in Confluence using all the power of AI and get smart and user-friendly responses. Moreover, you can create a page with content which can be prepared with AI as well, which is a very useful capability and saves time.

**Q: What functionalities does the Research tool offer?**  
A: The Research toolkit integrates the assistant with web search engines, supporting tools like Google Search Tool Json, Tavily Search Results Jason, Wikipedia, and Web Scrapper, to operate with the most current data.

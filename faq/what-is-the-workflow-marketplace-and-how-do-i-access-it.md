# What is the Workflow Marketplace and how do I access it?

The Workflow Marketplace is a centralized hub in AI/Run CodeMie where authenticated users can discover and execute globally published workflows. Unlike project-scoped workflows, marketplace workflows are available to all users and run using the publisher's project resources — datasources and assistants — so no personal configuration is required to execute them.

To access the Marketplace, navigate to **Workflows** → **Marketplace** in the left sidebar. From there, browse the catalog, filter by category, or search by name to find a workflow.

Each workflow card displays the number of unique users who have executed it. Clicking a card opens the workflow details page, where the workflow can be run directly or cloned to a personal project for customization.

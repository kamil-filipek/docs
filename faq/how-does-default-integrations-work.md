# How does default integrations work ? By what criteria is the default integrations selected? What happens if I don’t choose the required integrations myself? If the user does not explicitly select an integration, the system will automatically choose a **default integrations** based on the following priority:

1. **User Integration:**  
   A personal integration configured by the user for their own account.

2. **User Global Integration:**  
   A personal integrations marked as **global** (i.e., available across all projects for the user). This type allows users to reuse the same integrations in multiple contexts.

3. **Project Integration:**  
   A shared integration available at the project level. These are typically configured by users with elevated roles such as applications-admin. They are intended for collaboration and reuse within a team or project.

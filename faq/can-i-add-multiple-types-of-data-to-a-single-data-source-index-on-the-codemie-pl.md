# Can I add multiple types of data to a single Data Source index on the CodeMie platform?

No, each Data Source index is specific to one type of data. You must choose between Links, JSON file, or Text/PDF files when setting up your Data Source. However, you can upload up to 10 PDF/TXT files with maximum size of each up to 50MB.

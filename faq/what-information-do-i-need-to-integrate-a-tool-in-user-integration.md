# What information do I need to integrate a tool in User integration? Can I integrate multiple tools in CodeMie? How do I update the credentials for an integrated tool? How can I remove tool integrations from my User integration?

Most tools require you to provide a specific URL and a token for integration. Ensure you have these details ready before starting the integration process.  
You can integrate multiple tools through the User integrations menu. Follow the integration process for each tool you wish to add to expand your assistant's capabilities.  
To update the credentials for an integrated tool, navigate to the User integrations menu, find the tool you want to update, and click the "Edit" button. Here, you can update the URL, token, or any other necessary information and save the changes.  
To remove a tool integration, go to the User integrations menu, locate the tool you wish to remove, and click the "Delete" button. Confirm the deletion to remove the integrations from your assistant.

# How to use a microphone in CodeMie chat ? Voice input CodeMie speech to text? Talk to an assistant instead of typing? CodeMie voice recognition not working? How to Use Speech-to-Text?

CodeMie supports Speech-to-Text functionality that allows users to dictate their queries or commands verbally, converting spoken words into text for the assistant to process.

1. **Start a chat** with your desired assistant
2. **Locate the microphone button** in the right corner of the input field
3. **Click the microphone button** to activate voice recording
4. **Speak your question clearly** (maximum 10 seconds)
5. **Click Enter** to send your converted text message

Key Features

- **Automatic language detection**: The system automatically detects the language you're speaking
- **Real-time conversion**: Your speech is converted to text immediately
- **integrations with all assistants**: Works with any CodeMie assistant

Important Limitations:

- **10-second maximum** recording time per voice input
- **Browser compatibility**: Requires a modern browser with microphone support
- **Microphone permissions**: Browser must have microphone access enabled for the CodeMie platform

Technical Requirements:

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Microphone access permissions granted to the website
- Stable internet connection for speech processing

## Troubleshooting Guide:

**Q: I don't see the microphone button**

- Ensure your browser supports speech input
- Check if microphone permissions are granted to the website
- Try refreshing the page or using a different browser  
  **Q: The microphone button disappeared while typing**
- This is expected behavior - the microphone icon may not be visible when actively typing
- Clear the input field to make the microphone button reappear  
  **Q: My voice input isn't being recognized**
- Check your microphone is working in other applications
- Ensure you're speaking clearly and at normal volume
- Verify your internet connection is stable
- Try speaking in a quieter environment  
  **Q: I hit the 10-second limit**
- Break longer questions into shorter segments
- Speak more concisely and directly
- Use text input for complex or lengthy queries  
  **Q: The language detection is incorrect**
- Speak more clearly in your intended language
- Consider typing if the automatic detection consistently fails
- The system works best with clear pronunciation

Browser Permission Setup

1. **Chrome**: Click the lock icon in address bar → Allow microphone
2. **Firefox**: Click the microphone icon in address bar → Always Allow
3. **Safari**: Safari menu → Preferences → Websites → Microphone → Allow
4. **Edge**: Click the lock icon → Permissions → Allow microphone

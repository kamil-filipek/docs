# What is the recommended indexing mode and why?

The recommended indexing mode is "Whole codebase" because it provides a comprehensive overview of technology, libraries, stack and can be used for future tasks implementation using assistants.  
You can use “Summarization per file” if you want to generate documentation for the entire project.

# Is there any overview of the assistant templates available in CodeMie?

Yes, CodeMie offers various assistant templates:

- **Epic/User story compose:** Analyzes and generates project requirements, creates epics, user stories in Jira.
- **Coder** Generates implementation plans for tasks, accesses Jira, and implements changes in Remote Git.
- **Code Implementation Planner** Creates technical tasks from business requirements or stories, generates implementation plans, and creates technical tasks in Jira.
- **Code Reviewer:** Reviews changes in Pull Requests and creates comments on findings.
- **Google Search Assistant:** Conducts internet searches using Google, Wikipedia, Web Scraper, and Tavily search tools.
- **Project Onboarding Assistant:** This is useful when you want to ask questions about your codebase, Data Source or project specific questions..
- **ChatGPT:** Functions similarly to ChatGPT 4.0, with all typical capabilities except for processing images (this functionality is planned for the future).

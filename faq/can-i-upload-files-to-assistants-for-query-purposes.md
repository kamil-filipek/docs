# Can I upload files to assistants for query purposes?

Yes, you can upload files directly into the conversation with the assistant using the paperclip (📎) icon. The assistant will use the uploaded file to respond to relevant queries. Supported file types include jpeg, png, jpg, gif, pdf, pptx, docx, xlsx and csv. The assistant always prioritizes the latest uploaded file unless you explicitly reference a different file by name.

# How do A2A Remote Assistants differ from regular CodeMie assistants?

A2A Remote assistants are marked with an A2A label to differentiate them from regular CodeMie assistants. When working with remote assistants:

- Starting conversations: Click on the remote assistant card to open the chat interface (same as regular assistants)
- Chat features: Type questions, receive real-time status updates, view results, and continue multi-turn conversations with context
- View capabilities: Click the assistant icon in the chat to view the agent card and detailed capabilities at any time
- Limited editing: Remote assistants support limited editing to maintain integrity with the external agent
- Deletion: Can be deleted like regular assistants, but this only removes the reference in CodeMie—the external agent continues to run independently

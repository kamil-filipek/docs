# What is the purpose of JSON-RPC 2.0 in A2A Protocol?

JSON-RPC 2.0 protocol is used for reliable agent interactions in A2A. This is the standardized communication protocol that:

- Enables CodeMie to communicate with external agents
- Provides a common language for agent-to-agent interactions
- Ensures reliable message exchange over HTTP/HTTPS

All A2A-compatible agents must support JSON-RPC 2.0 protocol for communication.

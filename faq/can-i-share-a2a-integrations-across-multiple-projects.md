# Can I share A2A integrations across multiple projects?

Yes, when configuring A2A Integration, you can use the Global Integration setting:

- Toggle on Global Integration to use the integration across multiple projects
- If disabled, the integration will only be available within the selected project

This allows you to reuse authentication credentials for A2A agents across different projects in CodeMie.

# How to Install and Use the CodeMie VS Code Extension in IDE Visual Studio Code copilot? How do I use CodeMie in Visual Studio Code IDE in copilot?

## Overview

CodeMie integrates directly into Visual Studio Code through the GitHub Copilot Chat interface, providing access to AI-powered assistants within your development environment. This integration enables you to leverage CodeMie capabilities through the familiar Copilot Chat experience.

## Prerequisites

- **Visual Studio Code** installed on your system
- **CodeMie Account** for authentication

## Installation

1. Open Visual Studio Code
2. Access the Extensions panel:
   - Click the **Extensions** button in the Activity Bar, or
   - Press **Ctrl+Shift+X** (Windows/Linux) or **Cmd+Shift+X** (macOS)
3. Search for **"CodeMie"** in the marketplace
4. Click **Install**
5. Follow the setup instructions provided in the extension documentation

## Getting Started

After installation, the extension integrates with the GitHub Copilot Chat interface:

- Access CodeMie assistants through the Copilot Chat panel
- Interact with AI-powered assistants directly within your code editor
- Maintain your development workflow without switching contexts

## Troubleshooting

### Extension Not Responding

If the extension becomes unresponsive:

1. Click the **Refresh** button in the extension interface
2. Restart Visual Studio Code
3. Check your connection status in the extension settings

### Authentication Problems

If you experience authentication issues:

1. Verify your connection through the extension settings
2. Clear the extension cache:
   - Open VS Code settings
   - Search for "CodeMie"
   - Locate cache settings and clear cached data
3. Reset login data through the extension settings panel
4. Restart VS Code and authenticate again

### Additional Support

If issues persist after trying these solutions:

- Review the extension's output logs (View → Output → Select "CodeMie" from dropdown)
- Check the extension's documentation for updated troubleshooting steps
- Contact support through the CodeMie platform

# Will integrating tools in User integrations affect the performance of my assistant? Should I recreate the assistant if I forgot to add integrations and added it later?

Integrating tools should not significantly affect the performance of your assistant. However, the response time might slightly increase when performing tasks that require interaction with the integrated tools, depending on the tool's response time.  
You don’t need to recreate the assistant when you forgot to add integrations and added it later. Just continue using the same chat.

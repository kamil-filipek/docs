# What is streaming in A2A and how does it work?

Real-time Streaming in A2A allows you to get status updates and results as they're processed. When an A2A agent supports streaming:

- CodeMie receives real-time status updates during processing (e.g., "Processing...", "Working...")
- Users see progress messages in the chat interface
- The final result is delivered when processing completes

Streaming requires the agent to implement the task subscribe endpoint (for streaming). Not all agents support streaming—check the agent card or documentation for streaming capabilities.

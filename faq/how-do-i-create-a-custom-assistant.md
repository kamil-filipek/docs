# How do I create a custom assistant?

To create a custom assistant, follow these steps:

1. Go to the main page and click the **"Assistants"** button
2. Click "+ Create Assistant."
3. Fill in the assistant properties:
   - Name: Enter the name of your assistant.
   - Description: Provide a description for your assistant.
   - System Instructions: Specify a prompt that the assistant's answers will be based on.
   - Predefined Questions (Optional): Add up to 4 questions that will appear as conversation starters in the chat interface. Users can click these questions to quickly populate the input field. Examples include "Tell me your capabilities" or "Show me available Tools".
   - Icon URL (Optional): Provide the URL for an icon to represent your assistant.
   - LLM Model: Choose between GPT 4o for better response time and GPT 4 for better response quality.

- Context & Data Sources (Optional): Choose a Data Source for the assistant, like project documentation or a Git repository.
- Tools & Integrations : Enable tools to expand your assistant's capabilities. Each toolkit is represented as a separate expandable card with specific configuration options:
- Standard tools (Jira, Git, Research, etc.)
- External tools (MCP Servers)

Each tool card displays an icon representing the toolkit and provides configuration options specific to that tool.  
4. After creation, your assistant will be listed in the Assistants -> Project Assistants  
5. Use the dedicated buttons to edit or delete your assistant.  
6. Click the assistant’s name to start a conversation. Your assistant will also be added to the quick Assistants list.  
7. Test your assistant with complex tasks, such as generating a code plan for a project task.

# What Azure DevOps tools are available in CodeMie assistants?

CodeMie provides three Azure DevOps tools that can be added to any assistant:

- **Azure DevOps Work Items** — create, update, search (WIQL), link, and comment on work items (Task, Bug, Issue, Epic)
- **Azure DevOps Wiki** — read, create, modify, move, rename, search, and manage attachments on wiki pages
- **Azure DevOps Test Plans** — manage test plans, test suites, and test cases

All three tools use the same **AzureDevOps** credential type and share a single integration setup.

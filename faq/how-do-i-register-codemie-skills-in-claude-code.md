# How do I register CodeMie skills in Claude Code?

Run `codemie setup skills` in your terminal. The wizard walks you through three steps:

1. Read the disclaimer and press **Enter** to continue
2. Choose a scope — **Global** to use the skill across all projects, or **Local** for the current repository only
3. Browse available skills by tab (Project, Marketplace, or Registered) and press **Space** to select, **Enter** to confirm

Once registered, the skill is available in Claude Code as a slash command — type `/skill-name` to invoke it.

# Is it possible to customize the functionality of an assistant in CodeMie?

The assistants are based on predefined templates designed for specific roles. While customization options are not explicitly mentioned, using templates helps improve the response quality and achieve satisfactory results. You can edit the assistant using actions on the assistant's card.

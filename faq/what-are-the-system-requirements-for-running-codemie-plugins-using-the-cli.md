# What are the system requirements for running CodeMie plugins using the CLI?

```markdown
 What are the system requirements for running CodeMie plugins using the CLI?

To run CodeMie plugins using the CLI, you need:

- Python 3.12 or higher
- For best results, install `uv` package installer:
  - On MacOS/Linux: `curl -LsSf https://astral.sh/uv/install.sh | sh`
  - On Windows: `powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"`
- Node.js 22 or higher (for certain plugins and MCP servers)
## Sources

- [Plugin](https://docs.codemie.ai/user-guide/tools_integrations/tools/plugin)
```

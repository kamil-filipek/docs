# What file formats can I upload to the assistant in chat? Which types of files are supported for upload in this chat?

The CodeMie allowed file formats in chat assistant are:

- Images: jpeg, png, jpg, gif
- Documents: pdf, pptx, txt (plain text files)
- Data: csv

The maximum file size is 50MB.

When working with:

- CSV files, I can ask questions about how to display the information in a specific format or make selections based on certain criteria.
- Plain text files (like logs or code), I can analyze the content, search for patterns, or extract specific information.

Users can refer to the uploaded files from chat history. The system also supports pasting large blocks of text directly into the chat without limitations.

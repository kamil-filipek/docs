# Can I share an A2A Remote Assistant with my project team?

Yes, when creating an A2A Remote Assistant, you can configure sharing:

- Toggle on "Share with project" to make the assistant available to project team members
- This allows other team members in the project to use the remote assistant
- The assistant remains linked to the project you selected

This is useful for teams that want to collaborate using the same external A2A agents.

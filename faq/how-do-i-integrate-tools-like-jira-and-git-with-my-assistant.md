# How do I integrate tools like Jira and Git with my assistant?

To use Jira and Git tools with your assistant, you must first integrate them with CodeMie in the "User integration" menu or as an alternative way of getting to the User integrations page, you can click the New integrations button in front of the desired tool when creating/editing your assistant. This link will also lead to the New integrations page. Note that this link appears only if no such tools are configured by the users.If you have two or more integrations related to one tool, you can specify the needed one when adding/editing assistants. After integration, you can enable these tools during the assistant creation process to allow your assistant to interact with Jira issues and Git repositories.

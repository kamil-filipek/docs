# User integrations Overview. What is the User integrations menu? How can I integrate a tool using the User integrations menu?

The User integrations menu in CodeMie is a feature that allows users to integrate various tools to enhance the capabilities of their assistants. With these integrations, assistants can perform complex tasks beyond answering questions, such as creating pull requests for a Git repository, updating Jira issues, fetching the latest news from news sites, etc.

To integrate a tool, follow these steps:

1. In the CodeMie main menu, click the "integrations" button.
2. In the User integrations menu, click the "new integration" button.
3. Select the tool you wish to integrate.
4. Specify the required Alias and credentials (usually the URL and token).
5. Click "Create Integration" to finalize the integration.

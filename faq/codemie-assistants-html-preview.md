# CodeMie assistants html preview? Codemie assistant html view?

1. Open any assistant and ask them for example: "Can you provide me with an example of an HTML login page code?"
2. After the assistant generates the code, you can click on the “\</>” button in the top-right corner to see the preview of webpage.

# What environment variables are needed to run the A2A Currency Conversion Agent?

To run the A2A Currency Conversion Agent example, configure these environment variables:

- CHAT_MODEL_PROVIDER=azure (or your LLM provider)
- AZURE_OPENAI_API_KEY=your_api_key_here
- AZURE_OPENAI_ENDPOINT=your_endpoint_url
- AZURE_OPENAI_API_VERSION=2024-12-01-preview

Set these in a .env file in the agent directory before running the agent.

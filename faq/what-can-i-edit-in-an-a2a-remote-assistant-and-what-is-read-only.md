# What can I edit in an A2A Remote Assistant and what is read-only?

A2A Remote assistants support limited editing to maintain integrity:

Editable fields:

- Assistant name
- Project assignment
- Sharing settings

Read-only fields:

- All agent-specific fields (description, tools, capabilities)

This ensures that the remote agent's configuration remains consistent with its actual implementation. To change agent-specific settings, you must modify the agent itself and then re-fetch the agent card in CodeMie.

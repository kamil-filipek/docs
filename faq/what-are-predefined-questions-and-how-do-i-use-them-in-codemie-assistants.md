# What are predefined questions and how do I use them in CodeMie assistants?

What are predefined questions?  
Predefined questions are commonly asked questions that you can configure when creating or editing an assistant. These questions appear in the chat interface, allowing users to quickly select them without typing, enhancing efficiency and user experience.

How to add predefined questions to an assistant

1. When creating a new assistant or editing an existing one, locate the "Predefined Questions" section on the assistant configuration page.
2. You can add up to 4 predefined questions.
3. Enter questions that users might frequently ask, such as:
   - "Tell me your capabilities"
   - "Show me available tools"
   - "How to connect to a database?"
   - "Help me with code review"

How to use predefined questions in a chat

1. When you start a chat with an assistant that has predefined questions configured, these questions will appear in the chat interface.
2. Click on any of the predefined questions to automatically populate the input field with that question.
3. Press Enter or click the Send button to submit the question.
4. The assistant will generate an appropriate response to the selected question.

Benefits of using predefined questions

- **Efficiency**: Access common questions with a single click
- **Consistency**: Ensure users ask questions in the optimal format
- **Discoverability**: Help users understand what the assistant can do
- **Onboarding**: Guide new users to effective interaction patterns

Examples of good predefined questions

- For general assistants: "What can you help me with?", "Tell me about your capabilities"
- For code assistants: "Review this code for bugs", "Help me optimize this function"
- For research assistants: "Find information about [topic]", "Summarize the latest research on [topic]"
- For project management assistants: "Show my active Jira issues", "Create a new user story"

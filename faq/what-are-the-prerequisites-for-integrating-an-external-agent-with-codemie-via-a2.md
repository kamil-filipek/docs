# What are the prerequisites for integrating an external agent with CodeMie via A2A?

To integrate an external agent with CodeMie via A2A, you need:

- Agent URL: The external agent must expose an A2A-compatible endpoint
- Agent Card: The agent should provide a card describing its capabilities
- Network Access: CodeMie must be able to reach the agent's URL

Note: Most A2A agents are publicly accessible and don't require authentication. If your agent requires authorization, see the Authentication section in the A2A documentation.

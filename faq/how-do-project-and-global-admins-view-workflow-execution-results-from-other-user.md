# How do project and global admins view workflow execution results from other users? Can project admin see workflow results from other users? Why can't I see my team member's workflow execution results?

Project and global administrators in Codemie have enhanced visibility into workflow executions within their project scope, allowing them to monitor and oversee all workflow activities regardless of who triggered the execution.

## Project Administrators (attribute: application_admin)

- Can view execution results of all workflows triggered by any user within their project
- Have access to both shared and non-shared workflow execution results
- Can monitor workflow activity through the Execution History section in the codemie.lab portal

## Global Administrators (role: admin)

- Have the same visibility as project administrators across all projects
- Can view execution results from any user in any project they have global access to
- Maintain comprehensive oversight of workflow activities organization-wide
